# Fablr - Maak leuke verhaaltjes met Fable Fableton

Fablr is een speelse chatbot die basischool leerlingen help met het bedenken van leuke fantasierijke verhaaltjes.

---

## Inhoud
- Over
- Functionaliteiten
- Techische stack

---

## Over 
In deze tijd van technologie is het lastiger voor kinderen om hun fantasie te gebruiken, omdat het meeste al voor hun uitgekauwd is. Hierdoor moet deze chatbot er op een leuke manier voor zorgen dat ze hun hersenen stimuleren om hun fantasie te gebruiken en hun schrijfvaardigheid te verbeteren.
De chatbot beantwoord natuurlijk vragen maar hij stelt ook tegenvragen, zodat jij beter kun nadenken over het verhaal wat jij wilt creeren, want het doel is uiteindelijk dat jij het verhaaltje zelf schrijft.

---

## Functionaliteiten
 - Interface met teskt ballonetjes
 - Markdown tekst kan getoond worden
 - Token usage wordt bijgehouden
 - Submit button inactief tijdens verwerken prompt
